# make 命令
	make [为目标名称]
	make all # 执行 目标 all 之下的代码
	-f 后接需要解释执行的 makefile 的文件名

# 约定
	习惯上使用 all 目标编译所有文件
	习惯上使用 clean 目标清空所有的 .o 文件
	习惯上使用 install 目标用于将编译好的程序复制到安装目录下
	此安装目录是通过执行 configure 脚本通过 --prefix 参数配置的

# 目标的书写规则
目标名：目标所依赖的文件
	所需要执行的命令


# 系统变量
	变量引用方法 $(var_name)
	还有一些默认变量名

# 自动化变量
	$@ 代表当前所在区域的目标文件 也就是 冒号左边的那些文件
	$^ 代表当前所在区域的依赖文件 也就是 冒号右边的那些文件
	$? 代表当前所在区域的产生变化的依赖文件 也就是 冒号右边的那些文件的一个子集
	# ? 问好代表，哪些文件产生变化了？是不是有很多问号？
	# $ hat -> $^
	$< 代表依赖文件列表中的最左边的那个文件的字符串
	$> 代表依赖文件列表中的最右边的那个文件的字符串
